# Supplementary Code - Giner et al. (2025)

R scripts for the analysis presented in:
**"Steroid Phenotype Stratification Reveals Distinct HLA Expression Signatures in Adrenocortical Carcinoma"**, published in *Cancers* (MDPI).
**DOI:** 10.3390/cancers18020229

## Structure
The repository is organized by cohort. Scripts should be run sequentially (01, 02, ...).

* **01_TCGA_PANCAN**: Global HLA patterns analysis.
* **02_TCGA_ACC**: Main Adrenocortical Carcinoma analysis.
* **03_ENSAT_ACC**: External validation (GEO: GSE49278).
